# retirement-pro

Retirement Planning Dashboard for tracking savings, projections and goals.

## Tech Stack

React 18 + Vite + TypeScript + Tailwind CSS + Recharts + Zustand

## Dev Commands

| Command | Purpose |
|---------|---------|
| `npm run dev` | Dev server |
| `npm run build` | Production build |
| `npm run lint` | Lint |
| `npm run typecheck` | Type check |
